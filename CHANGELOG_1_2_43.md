# ELTA ITS v1.2.43

- La tarjeta **Alertas activas** del dashboard ahora muestra únicamente alertas sin verificar asociadas a **tránsitos abiertos**.
- Se excluyen alertas de tránsitos finalizados/cerrados/cancelados del contador principal.
- La campana y el contador superior usan el mismo criterio: alertas pendientes de tránsitos abiertos.
- El listado compacto del dashboard muestra solo las últimas alertas pendientes de tránsitos abiertos.
